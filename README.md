# Анализ оттока клиентов банка (Churn Modelling)

## Описание проекта
Цель проекта — выявить ключевые сегменты клиентов, склонных к уходу из банка.
Использован комбинированный подход: обработка данных с помощью SQL и статистический анализ на Python.

## Стек
* SQL: Оконные функции, фильтры, CTE.
* Python: Pandas, Matplotlib, Seaborn.
* Статистика: Коэффициенты корреляции Пирсона, Спирмена, Кендалла.

## Выводы
1.  Продуктовая аномалия: Клиенты с 3 и 4 продуктами уходят в 83% и 100% случаев. Это критическая точка риска.
2.  География: Германия имеет уровень оттока 32%, что вдвое выше показателей Франции и Испании.
3.  Возрастной фактор: Основная группа оттока — клиенты старше 40 лет. 
4.  Статистика: Ранговые корреляции (Спирмен/Кендалл) подтвердили нелинейную связь между количеством продуктов и оттоком, которую не зафиксировал стандартный метод Пирсона.

## Визуализации
В проекте представлены:
* Матрицы корреляций.
* Распределение оттока по странам и возрасту.
* Отток по количеству продуктов.
